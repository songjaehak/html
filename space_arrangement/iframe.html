HTML iframe

HTML iframe이란 inline frame의 약자로, 웹 페이지안에 또 다른 웹페이지를 삽입하는 것

사용법

<iframe src="삽입하는 웹페이지" title="내용"></iframe>

  • title: 웹 브라우저에 해당 iframe이 어떤 iframe인지 알린다. 이 속성은 필수는 아니지만 권장한다.
width, height 속성 사용

width, height 속성을 사용하면, 너비, 높이를 지정가능

<iframe src="삽입하는 웹페이지 URL" title="내용" width="100%" height="300px"></iframe>

웹 페이지 안에 위크독스 웹 페이지를 삽입 시키는 예제
<h3>iframe을 이용하여 웹 페이지에 다른 웹 페이지 삽입하기</h3>
<iframe src="https://wikidocs.net/" title="내용" width="50%" height="300px"></iframe>

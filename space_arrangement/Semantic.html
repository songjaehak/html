HTML Semantic

HTML5에서 추가된 의(Semantic) Tag

  • <header> : HTML 헤더나 section 헤더를 정의
  • <nav> : HTML 메뉴바를 정의
  • <main> : HTML 전체 Content를 정의
  • <section> : HTML Section을 정의
  • <article> : HTML 하나의 글(article)을 정의
  • <asdie> : 웹 페이지에서 배너나 추가적인 문구 등을 정의
  • <footer> : HTML footer를 정의

예시

header 부분입니다.

nav 부분입니다.

aside 부분입니다.

footer 부분입니다.

블록(block) 형태

블록 형태의 속성을 가진 tag들은 네모 블록 모양을 가짐.
항상 새로운 줄에서 시작하며, 해당 줄의 모든 너비 차지

<div> tag

  • 다른 HTML Tag들을 하나로 묶는데 사용
  • 하나로 묶인 다른 HTML Tag에 CSS 스타일을 한번에 적용하는데 자주 사용합니다
예시 
<div style="border:1px solid black;">
  <h1 style="text-align:center">HTML5 & CSS3</h1>
  <p>HTML5와 CSS3를 공부하자</p>
</div>

HTML5 & CSS3

인라인(Inline) 형태

새로운 줄에서 시작하지 않으며, 해당 줄의 너비는 내용만큼만 가집니다.

<sapn> Tag

  • 텍스트의 특정부분을 강조하는데 자주 쓰임
  • 텍스트의 특정부분에 css 스타일 적용을 위해 자주 사용
예시

<p> 우리는 <span style="color:red">HTML5 & CSS3</span>를 공부하고 있습니다.</p>

>> 우리는 HTML5 & CSS3를 공부하고 있습니다.

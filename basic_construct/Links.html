속성 의미 예시
_self 디폴트 값으로서, 현재 창에서 해당 주소 문서 창/탭 이 열린다. 위키독스
_blank 현재의 창이 아니라 새 창/탭에서 해당 주소 문서를 연다 위키독스
_parent 현재보다 상위 프레인에서 해당 주소 문서를 연다 위키독스
_top Root 프레임에서 해당 주소 문서 위키독스

State Explanation
link 한번도 방문한적 없는 상태(default), 파란색을 띔

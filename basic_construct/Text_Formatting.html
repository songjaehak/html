Tag Explanation Example
<b> 텍스트를 굵게 출력 텍스트가 굵어짐
<strong> 텍스트를 굵게 + 중요함 의미 b tag와 비슷하지만 의미적으로 더 중요함을 의미
<i> 텍스트를 이탤릭체로 출력 이탤릭체로 출력. HTML5에서는 css3을 이용해 서식 변경 가능
<em> 텍스트를 이탤릭체 + 중요함 의미 i tag와 비슷하지만 의미적으로 더 중요함을 의미
<small> 텍스트 폰트 크기를 작게한다. 얼마나 작아지는 걸까요...?
<sub> 첨자 텍스트 seqeunce Xn
<sup> 위첨자 텍스트 e2 이에 이승
<ins> 삽입된 텍스트에 밑줄을 긋는다. ins는 insert의 약자로 밑줄을 그어줍니다.
<del> 텍스트에 취소선을 긋는다. del을 delete의 약자로 취소선을 그어줍니다.
<mark> 텍스트에 형광펜처럼 강조표시 형을 무척 좋아하는 동생은? 형광펜

순서가 없는 리스트

사용법 
<ul>
    <li>항목 1</li>
    <li>항목 2</li>
    <li>항목 3</li>
</ul>

<ul>: unordered list의 약자

예시
  • HTML5
  • CSS5
  • Java script

style="list-style-type"을 이용해 점 모양을 바꿀 수 있다

  • disc : 디폴트 값으로 검은색 점이다.
  • circle: 원
  • square: 사각형
  • none: 아무것도 표시되지 않는다.
예시 : list-style-type:square
  • HTML5
  • CSS3
  • Java Scirpt

순서가 있는 리스트

사용법 
<ol>
    <li>항목 1</li>
    <li>항목 2</li>
    <li>항목 3</li>
</ol>

<ol>: ordered list의 약자

예시
  1. HTML5
  2. CSS5
  3. Java script

type을 이용해 다양한 모양으로 변경 가능

  • 1 : 디폴트, 아라비아 숫자
  • A : 대문자 알파벳
  • a : 소문자 알파벳
  • I : 대문자 로마숫자
  • i : 소문자 로마숫자
예시 : <ol type="A">
  1. 항목 1
  2. 항목 2
  3. 항목 3

start 속성을 이용해 원하는 시작 숫자로 바꿀 수 있음.

>예시 : <ol start="30">
  1. 항목 1
  2. 항목 2
  3. 항목 3

정의 리스트(definition)

사용법 
<dl>
<dt>용어 1</dt>
<dd>용어 1에 대한 정의</dd>
<dt>용어 2</dt>
<dd>용어 2에 대한 정의</dd>
</dl>

<dl> :definition list의 약어이며, 정의리스트를 정의합니다.
<dt> : definition Term의 약어, 용어를 입력해줍니다.
<dd> : definition Description의 약어, 용어에 대한 정의를 해줍니다

HTML5

CSS3
Cascading Style Sheets

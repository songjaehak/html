HTML에서 이미지를 삽입 하려면 <img> 태그를 사용해야함

사용법 : <img src="url" alt="대체 텍스트">

절대경로와 상대경로
  • 절대경로

    • 고유경로

  • 상대경로

    • 절대경로와는 다르게 최상위 디렉토리 포함x
    현재 위치한 곳을 기준으로 해당파일이 있는곳 까지 감

    • /: 최상위 디렉토리로 이동
    • ./: 현재 위치한 디렉토리를 의미
    • ../: 상위 디렉토리로 이동
width 속성, height 속성

사용법 : <img src="url" alt="대체 텍스트" width="숫자" height="숫자">

이미지에 링크 설정

사용법 : <a href="URL"><img src="URL" alt="대체 텍스트" width="숫자" height="숫자"></a>

HTML input

다양한 타입의 input tag를 사용하여 사용자로부터 입력 받을 수 있다.

  1. 텍스트(text)
  2. 패스워드(password)
  3. 라디오 버튼(radio)
  4. 체크 박스(check)
  5. 파일 선택(file)
  6. 텍스트 박스(textarea)
  7. 선택입력(select)
  8. 버튼(button)
  9. 전송(submit)
  10. 필드셋(fieldset)
